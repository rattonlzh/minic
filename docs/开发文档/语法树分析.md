语法树用广义表来描述:表名(表项1, 表项2, ...),表项也可以是一个广义表, 表名即语法树的根节点,表项是语法树的子树节点
分析BNF得出语法树的结构如下:
```
程序组成:变量声明 函数声明
- 程序名(变量, 函数)
- 标识符(变量类型, 值)
- 函数标识符(返回类型, 参数列表, 复合语句)
- 参数列表(变量1, 变量2, ...)
- 复合语句(变量, 语句列表)
- 语句列表(语句1, 语句2, ...)
- op (左表达式, 右表达式), op可以是=,+,-,*,/,==,!=, >, >=, <,<=
- [](标识符, 下标)
- call(函数标识符)
- if(分支条件, 条件成立时执行的语句, 不成立时执行的语句)
- while(循环条件, 复合语句)
```
数据结构选择使用多叉树, 一个树节点的孩子节点数是不定长的, 一个节点应具有以下信息:数据,保存孩子指针的链表
数据选择使用字符串类型
